Registro de muestras del Instituto Oceanográfico de Baleares

Para registrar el muestreo a partir de un archivo de audio el formato del archivo deber ser WAV o MP3. Puede pulsar este enlace para realizar la conversión.

¿Cómo desea realizar el muestreo?

Para realizar el registro desde el archivo deber tener el micrófono funcionando y el altavoz puesto al máximo.
